# 2020 제 1회 CAU 응용통계학과 분석 공모전

# 1. 개요
- 주제: TV 프로그램이 골목 상권에 미치는 영향 분석 및 예측
- 수상: 우수상 (16개 팀 중 3위)
- 기간: 2020.09 ~ 2020.11
- 팀원: 김성관, 박준근, 정서연, 황재원
- 사용 skill: Python, R
